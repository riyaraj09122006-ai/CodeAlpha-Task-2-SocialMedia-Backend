# CodeAlpha Task 2 - Social Media Backend API

## ✅ Task Status: Completed

## Tech Stack
Node.js, Express.js, MongoDB, JWT, bcryptjs

## Features Implemented
1. User Registration with Password Hashing
2. User Login with JWT Token Generation
3. JWT Authentication Middleware 
4. Protected Route: Create Post API
5. MongoDB Database Integration

## API Endpoints
| Method | Endpoint | Description | Auth Required |
| --- | --- | --- | --- |
| POST | /api/users/register | Register new user | No |
| POST | /api/users/login | Login & get token | No |
| POST | /api/posts | Create new post | Yes |

## Testing
Successfully tested using Thunder Client.
Result: Status 200 OK. Post saved to database with user ID and timestamp.
